## Telepítés:

[Node.js](https://nodejs.org/en) minimum `18.0.0` verzió szükséges a projekt futtatásához.

### PNPM telepítése:

```bash
npm install -g pnpm
```

### Függőségek telepítése:

```bash
pnpm install
```

### Dev szerver indítása:

```bash
pnpm dev
```
